
# AllstoreZA Product Review, Strategy & Roadmap

## Product Positioning
AllstoreZA is an AI-powered personal fitting room that helps users gain confidence before purchasing clothing online.

Core value proposition:
See how clothing might look on you before you buy.

## Key Findings
- The current VTO experience is the highest priority issue.
- Shopping should be the primary focus; AI should support the journey.
- One product per thumbnail/card.
- Guest mode is sufficient for Phase 1.
- Affiliate/deep-link model is preferable to a full marketplace.

## VTO Recommendation
Evaluate an image-generation workflow using:
- Avatar image
- Product image

Output:
- Generated image showing the avatar wearing the selected garment.

Benefits:
- Faster implementation
- Easier debugging
- Better reliability

## Homepage Improvements
- Reduce avatar dominance.
- Increase product visibility.
- Add search.
- Add filters.
- Add cart.
- Add wishlist.

## Product Card Rules
- One garment per card.
- No grouped thumbnails.
- Consistent image ratios.
- Buttons must never wrap:
  - Try-On
  - Cart
  - Wishlist
  - Delete

## Store Management
- Remove duplicate Manage Stores button.
- Keep a single button in the top-right of the catalog area.
